HTML Concepts

HTML (HyperText Markup Language) is the foundation of web pages. It defines the structure and content of a web page using a series of elements represented by tags.

HTML Structure

A basic HTML document consists of:

  • <!DOCTYPE html>: Declares the document as HTML
  • <html>: The root element of the document
  • <head>: Contains metadata (title, styles, scripts)
  • <body>: Contains the visible content of the page

HTML Selectors

Selectors are used to target specific HTML elements for styling with CSS or manipulation with JavaScript. Common selectors include:

  • Element selectors (e.g., p)
  • ID selectors (e.g., #main-header)
  • Class selectors (e.g., .error)

HTML Tags

HTML tags define the various elements of a web page. Here are some common tag types:

Text Formatting:

  • <h1> to <h6>: Headings
  • <p>: Paragraphs
  • <strong>: Strong emphasis
  • <em>: Emphasis

Lists:

  • <ul>: Unordered list
  • <ol>: Ordered list
  • <li>: List item

Links and Images:

  • <a href="">: Hyperlink
  • <img src="" alt="">: Image

Tables:

  • <table>: Table
  • <tr>: Table row
  • <th>: Table header
  • <td>: Table data

Forms:

  • <form>: Form
  • <input>: Text field, password field, etc.
  • <button>: Button
