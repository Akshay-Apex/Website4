CSS Concepts

Font Properties

CSS provides various properties to control font styles:

  • font-family: Specifies the font
  • font-size: Sets the font size
  • font-weight: Sets the font weight (e.g., bold)
  • font-style: Sets the font style (e.g., italic)
  • text-decoration: Adds text decorations (e.g., underline)
  • color: Sets the text color

CSS Styles

CSS can be applied in different ways:

Internal CSS

Styles defined within the <style> tag in the <head> section.

Inline CSS

Styles applied directly to elements using the style attribute.

Example of inline CSS

External CSS

Styles defined in a separate file (e.g., styles.css) and linked using the <link> tag.

CSS Selectors

Selectors target specific elements to apply styles:

  • Element selectors (e.g., p)
  • ID selectors (e.g., #main-header )
  • Class selectors (e.g., .error )
